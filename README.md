This is a project involving the full production process of Super Mario, written by python and uploaded to github using git.

第一次上传新项目到 GitHub

第 1 步：在项目目录初始化 Git
git init

第 2 步：添加所有文件到暂存区
git add .

第 3 步：提交到本地仓库
git commit -m "第一次提交"

第 4 步：关联远程仓库（GitHub 仓库地址）
推荐使用 SSH 地址
git remote add origin git@github.com:你的用户名/仓库名.git

第 5 步：推送到 GitHub
git push -u origin main
